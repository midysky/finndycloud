# 发源地开源云采集引擎

发源地云采集引擎是由发源地研发团队开发的一套开源云采集工具化引擎，支持本地化私有化部署，可以快速搭建属于自己的大数据云采集系统。

### V1.0功能列表
- 数据源的发布和管理：创建，复制，导入，导出，运行，调试等；
- 采集数据的预览，导出，API调用；
- 网站设置：基本信息，云帐号设置，SEO信息；
- 会员管理：新增，编辑等。

### 安装说明
>nginx下配置WWW主机域名指向站点下的【根目录】或【public目录】（更安全）。

>以下目录需要可写操作权限

- runtime/
- data/
- app/
